About Notepad

Notepad is a basic text editing program and it is most commonly used to view or edit text files. A text file is a type typically identified by the .txt file name extension

How do I change the font style and size?

Changes to the font styles and size affect all the text in the document.
Click the format menu and then click font.
An example of how your font will look appears under Sample.
When you are finished making fonr selecions, click Ok
